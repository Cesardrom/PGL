## Unidad 1

#### 多Que es un Programa?

    Un programa es un conjunto de instrucciones que se ejecutan en un dispositivo. El main de un proyecto es solo un metodo que lanza este programa final

#### 多Que es un Proceso?

    Un proceso es el programa en ejecucion que esta identificado y tiene diferentes caracteristicas. El proceso es una entidad activa y el programa es pasiva.
    Ademas todos los procesos son independientes el uno del otro ya que se crea uno al ejecutar un programa.

#### 多Que tipos de programacion hay?

    Tenemos diferentes tipos de programacion, dependiendo de la capacidad del procesador(nucleos) o lo que necesitemos para cada objetivo:
    - *Programacion Concurrente*: Los procesos que se ejecutan y se crean dependen entre si para ejecutar correctamente todo.
    - *Programacion Paralela*: Los procesos que se ejecutan y se crean son independientes y se ejecutan a la vez.
    - *Programacion Secuencial*:Se debe ejecutar solo un proceso a la vez y se debe terminar el anterior para ejecutar el siguiente.
